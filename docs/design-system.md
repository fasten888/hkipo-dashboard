# 港新账本 UI 设计说明

## 产品定位

港新账本从传统后台页面升级为个人投资驾驶舱，核心定位是 Personal Investment OS。页面优先回答三个问题：

1. 赚了多少钱
2. 风险在哪里
3. 下一步应该做什么

后续扩展美股 IPO、ETF、股票持仓、AI 分析和多账户资产时，继续沿用同一套信息架构。

## 视觉语言

- 背景：`#F7F9FC`
- 卡片：纯白、`20px - 24px` 圆角、轻边框、轻阴影
- Primary：`#2563EB`
- Success：`#22C55E`
- Purple：`#8B5CF6`
- Orange：`#F59E0B`
- Red：`#EF4444`
- Neutral：Slate

整体避免后台管理系统风格，参考 Linear、Vercel、Stripe、Raycast、Arc、Notion 与 Apple Human Interface。

## 字体与数字

- 主字体：Inter，兼容 SF Pro、PingFang SC、Microsoft YaHei
- 标题：Medium
- 正文：Regular
- 数字：SemiBold
- 重点金额使用更大的字号和更紧的字距，但避免全部数字超粗。

## 组件规范

### 卡片

统一使用 `os-card` 和 `os-card-hover`：

- 背景白色
- 轻边框
- 轻阴影
- Hover 上浮 `translateY(-2px)` 并增强阴影

### 按钮

统一为四种：

- Primary：蓝色主按钮
- Secondary：白色按钮
- Danger：红色危险操作
- Ghost：透明轻按钮

### 输入框

统一高度不低于 `44px`，圆角 `14px - 16px`，Focus 使用蓝色 Glow。

### 弹窗

弹窗统一大圆角、轻遮罩、顶部固定标题区，移动端保留全屏录入能力。

## 总览页信息架构

1. 核心 KPI：总收益、总收益率、总成本、总中签率
2. 次级指标：打新胜率、本月收益、暗盘收益、首日收益
3. 趋势与构成：收益折线图、收益构成圆环图
4. AI 建议预留：未来接入 AI 的决策位
5. Upcoming IPO：近期可申购、上市和资金释放节点
6. 账户 Command Center：Top5 账户卡片
7. 资金占用：融资、现金、待释放、持仓和可抵押额度
8. 最近动态：Activity Timeline
9. 风险与机会：账户、融资方式、行业与卖出策略的结论

## 页面统一原则

- 总览：从记录展示升级为投资驾驶舱
- 账户管理：盈利分析中心，不是账户通讯录
- 出入金与换汇：强调真实收益和汇率损益
- 持仓：强调市值、抵押率和打新能力
- 新股资料：强调节奏、行业和参与结构
- 申购记录：以申购记录为核心，支持高频批量操作
- 中签管理：强调命中率结构
- 卖出记录：强调净利润、佣金、单签收益和破发
- 数据统计：从表格排行转向策略复盘
- 云同步：强调本地与云端状态可解释

## 响应式

- Desktop：4 列 KPI，组合式分析模块
- iPad：2 列卡片
- Phone：1 列为主，关键 KPI 支持 2 列快速看板

所有移动端交互按钮保持至少 `44px` 高度。
